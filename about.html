Welcome to Planet Detroit Resource Hub

The hope for this site is that it will house links to relevant information for journalists, activists, academics, lawyers and anyone interested in learning more about the environment from websites, studies, and data.

Instructions for use

Getting Data

Under the ‘Datasets’ tab is the data catalog. Here you can search by title and description keyword, select specific ‘categories’, or select data sets from a particular organization

Alternatively, under the ‘Organizations’ tab, you can browse organizations, and on the organization's page, you can search all their related datasets. 

Adding Data

Before starting make sure you're logged in to Github and have access to edit the repository. If you don’t reach out to Planet Detroit, and either ask for access or have us add the data for you.

On this site, data is linked out to its original source or anywhere else it happens to be hosted including GitHub, DropBox, or Google Drive. Before adding data be sure to have it hosted somewhere and get the link. If your data set comes in multiple forms they should each be linked separately. 

If your data is from an existing Organization: 

  1. Under the ‘Datasets’ tab click the ‘Add Dataset’ button
  2. Fill in as much of the form as you can
    1. For each link add a new ‘Resource” by clicking the button
    2. Be sure to select as many categories as you think are relevant
  3. Click submit! The data should be visible in the catalog in a few minutes 

If your data is not from an existing Organization: 

  1. First, add a new organization 
    1. under the ‘Organizations’ tab click ‘Add Organization’ or the ‘Add Dataset’ page click the ‘+’ button.
    2. Fill in as much of the form as you can
      • The logo should be a link to an image
    3. Click submit! The organization should be visible in the catalog in a few minutes 
  2. Go back to the ‘Add Dataset’ page and use the tips above to fill out the form.
